# MoveNet 자세 분석 AI 모델 - 프로젝트 분석 가이드

## 📋 프로젝트 개요

**프로젝트명**: MoveNet 자세 분석 AI 모델  
**모델 타입**: TensorFlow.js 기반 실시간 자세 감지  
**주요 기능**: 웹캠을 통한 실시간 17개 신체 키포인트 감지 및 시각화  
**플랫폼**: 웹 브라우저 (크로스 플랫폼)  

## 🗂️ 파일 구조 및 역할

### 핵심 파일들
- **`index.html`**: 메인 웹 애플리케이션 (219줄)
  - 웹캠 연결 및 실시간 자세 분석 UI
  - TensorFlow.js 모델 로드 및 추론
  - 키포인트 시각화 및 성능 모니터링

- **`model.json`**: MoveNet 모델 구조 정의
  - 입력: `[1, 192, 192, 3]` (배치, 높이, 너비, RGB)
  - 출력: `[1, 6, 56]` (배치, 최대 6명, 키포인트 정보)
  - 데이터 타입: 입력 int32, 출력 float32

- **`group1-shard1of3.bin`**: 모델 가중치 파일 1
- **`group1-shard2of3.bin`**: 모델 가중치 파일 2  
- **`group1-shard3of3.bin`**: 모델 가중치 파일 3

## 🤖 AI 모델 기술 사양

### 모델 아키텍처
- **기반 모델**: MoveNet Lightning v1
- **백본**: MobileNet 기반 경량화 CNN
- **입력 크기**: 192×192 픽셀
- **출력**: 17개 신체 키포인트 (x, y, confidence)
- **다중 인물**: 최대 6명 동시 감지

### 키포인트 구조 (17개)
```
0: 코, 1-2: 눈, 3-4: 귀, 5-6: 어깨, 7-8: 팔꿈치
9-10: 손목, 11-12: 엉덩이, 13-14: 무릎, 15-16: 발목
```

### 데이터 처리 파이프라인
```
웹캠 영상 → 192×192 리사이즈 → 정규화 → int32 변환 → AI 추론 → 키포인트 시각화
```
